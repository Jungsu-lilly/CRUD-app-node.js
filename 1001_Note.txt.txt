Node.js 는 웹서버로서 동작하고 있다
사용자가 요청(접근)할 때마다 우리가 읽어들여야 할 파일을 만들게 됨.
그리고 그것을 읽음.

 < URL >
ex) http://opentutorials.org:3000/main?id=HTML&page=12
http : protocol (웹 브라우저와 웹 서버의 통신 규칙)
opentutorials.org : host(domain). 인터넷에 연결되어 있는 특정한 컴퓨터를 가리키는 주소
300 : port (어떤 서버와 통신할 것인지?)
main: path (컴퓨터 안의 디렉토리)
id=HTML&page=12 : query string (쿼리 스트링을 변경하면, 다른 페이지 로드. 웹 서버에게 데이터를 전달.
시작은 물음표. 값과 값은 & 값의 이름과 값은 = 로 구분.

<query String 의 값에 따라서 node.js를 통해 만든 웹서버가 사용자에게 동적으로 생성하는 정보를 전송하는 방법.>
